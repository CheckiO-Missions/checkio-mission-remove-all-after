Since you have to end up taking i-s after the border, you may just add each i into result with no doubt. Use list.append() method for this purpose. 

result = []
for i in items:
    result.append(i)
    ...

How to perform checking values and break iteration, if I already added border into result?

It's that simple, as you said!) You already added current i into the result. Now, using if statement, check if i equals border with == operator. If so, use break command to stop iteration. 

result = []
for i in items:
    result.append(i)
    if i == border:
        break
...
So, how my final solution should look like?

It's almost all. You just need to wrap your code into a function expression and return the list you've filled - result.

Here is the whole solution:

from typing import Iterable


def remove_all_after(items: list[int], border: int) -> Iterable[int]:

    result = []
    for i in items:
        result.append(i)
        if i == border:
            break
            
    return result

You may try to "optimize" the code above and put return result inside if statement instead of break. But in case border is not in items or items is empty at all, your function returns nothing (None)!

This solution looks like previous a bit, but uses generator function. It yields each value "outside" function and stops, when border was the last yielded. 

from typing import Iterable


def remove_all_after(items: list[int], border: int) -> Iterable[int]:

    for i in items:
        yield i
        if i == border:
            break

In this solution return a slice of items including border after finding its index or the whole items. 

from typing import Iterable


def remove_all_after(items: list[int], border: int) -> Iterable[int]:

    return items[:items.index(border)+1] if border in items else items

Why list.index() method did not raise an exception here? Because, despite of the written order of commands, if border in items actually executes first!
