Я уявлення не маю, як розпочати розв'язання цієї місії.

Ідея у тому, щоб створити новий список, проходити по items і додавати кожне значення у новий список. Після кожного додавання, перевіряти чи останнє додане значення рівне border. Якщо так - припинити ітерацію і повернути той список, що ти заповнив.

Добре, то з чого мені почати?

Першим кроком проголоси змінну result з початковим значенням у вигляді порожнього списку. Відразу після цього ітеруй списком items, використовуючи цикл for. 

result = []
for i in items:
    ...
Що робити далі? Мені потрібна підказка.

Оскільки ти маєш завершити взяття елементів i після border, ти поки можеш просто додавати кожен елемент i в result без жодних сумнівів. Використай метод list.append() для цього. 

result = []
for i in items:
    result.append(i)
    ...

Як здійснити перевірку значень і припинення ітерації, якщо я вже додав border в result?

Це так просто, як ти щойно сказав!) Ти вже додав поточний елемент i в result. Тепер, використовуючи вираз if, перевір чи i рівне border з оператором ==. Якщо так, використай команду break, щоб припинити проходження списком. 

result = []
for i in items:
    result.append(i)
    if i == border:
        break
...
То як має виглядати моє остаточне рішення?

Це майже все. Тобі лише потрібно обгорнути твій код у вираз функції і повернути список, що ти заповнив - result.

Ось повне рішення:

from typing import Iterable


def remove_all_after(items: list[int], border: int) -> Iterable[int]:

    result = []
    for i in items:
        result.append(i)
        if i == border:
            break
            
    return result

Тобі може спасти на думку "оптимізувати" код вище і помістити return result всередину виразу if замість break. Але у випадку, коли border не виявиться в items або items взагалі порожній, твоя функція нічого не поверне (None)!

Звісно, тримай!

Цей розв'язок виглядає трохи як попередній, але використовує функцію generator. Вона видає кожне значення "назовні" функції і припиняє роботу, коли border було останнє видане значення. 

from typing import Iterable


def remove_all_after(items: list[int], border: int) -> Iterable[int]:

    for i in items:
        yield i
        if i == border:
            break

У цьому розв'язку повертай зріс items включаючи border після знаходження його індексу або цілий список items. 

from typing import Iterable


def remove_all_after(items: list[int], border: int) -> Iterable[int]:

    return items[:items.index(border)+1] if border in items else items

Чому метод list.index() не викликав виключення тут? Тому що, незважаючи на написаний порядок команд, if border in items насправді виконується першою!
