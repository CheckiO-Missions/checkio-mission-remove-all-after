Nie wszystkie elementy są ważne. To, co należy zrobić, to usunąć wszystkie elementy po danym elemencie z sekwencji.

Na przykład, mamy sekwencję [1, 2, 3, 4, 5] i musimy usunąć wszystkie elementy, które następują po 3 - czyli 4 i 5.

Mamy tutaj dwa przypadki brzegowe:

  • jeśli nie można znaleźć elementu tnącego, sekwencja nie powinna zostać zmieniona;
  • jeśli sekwencja jest pusta, to powinna pozostać pusta.
